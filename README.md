Face Detection using OpenCV

A Python-based Face Detection project built using OpenCV and Haar Cascade classifiers.
This project demonstrates how to detect human faces in images using classical computer vision techniques.

📌 Overview

Face detection is a fundamental task in computer vision. This project uses pre-trained Haar Cascade models provided by OpenCV to identify and locate faces in an image.

✨ Features

Detects human faces in images

Uses Haar Cascade classifier

Simple and beginner-friendly implementation

Implemented using a Jupyter Notebook

🛠️ Tech Stack

Python

OpenCV

Computer Vision

⚙️ How It Works

Load the Haar Cascade XML file

Read the input image

Convert the image to grayscale

Detect faces using detectMultiScale()

Draw bounding boxes around detected faces

▶️ How to Run the Project
Step 1: Clone the Repository
git clone https://github.com/yourusername/Face-Detection-project.git
Step 2: Install Dependencies
pip install opencv-python

Step 3: Run the Notebook
jupyter notebook


Open Face Detection with OpenCV-Python.ipynb and run the cells.

📷 Output

The program detects faces in the input image and highlights them using rectangular bounding boxes.
